Drawing Pad 🎨
A lightweight, interactive web-based drawing application built with p5.js and the HTML5 Canvas API. This tool allows users to create freehand sketches, geometric shapes, and gradient-filled designs directly in their browser.

✨ Features
Freehand Drawing: Smooth "Pencil" tool for sketching.

Geometric Shapes: Dedicated tools for drawing Lines, Rectangles, and Circles.

Gradient Support: Toggle linear gradients for shapes, allowing transitions between two customizable colors.

Customization: * Change stroke and fill colors via a color picker.

Adjust stroke weight (2px to 200px) for bold or fine lines.

Eraser & Clear: Easily fix mistakes with the Eraser tool or wipe the entire canvas clean to start over.

Responsive Canvas: Automatically fits the width and height of your browser window.
